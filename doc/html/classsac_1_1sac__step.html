void sac::sac_step::operator() ( double &  t0,
state_type &  x0,
b_control &  u1 
)
inline

Performs an iteration of SAC control.

Parameters
[in,out]t0initial time associated with state vector input x0. This get updated by one sample time to t0=t0+ts after stepper completes.
[in,out]x0initial state vector to be integrated forward after one iteration of SAC. The stepper integrates x0 from time t0 to time t0+ts based on SAC controls.
[in,out]u1default control value to apply from t0 to t0+calc_tm. The stepper computes the new SAC control to appy from t0+calc_tm to t0+calc_tm+ts and returns it in u1.
